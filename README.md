# Todo CLI 工具 📝

一个简单易用的命令行待办事项管理工具，帮助你高效管理日常任务。

## ✨ 功能特性

- ✅ **添加任务**：快速记录新的待办事项
- 📋 **查看列表**：清晰展示所有任务状态
- 🎉 **完成任务**：标记任务为已完成（支持批量操作）
- 🗑️ **删除任务**：移除不需要的任务
- 🧹 **清理已完成**：一键清除所有已完成的任务
- 💾 **自动保存**：数据本地存储（JSON格式），不会丢失

## 🚀 快速开始

### 前提条件

- Python 3.x

### 安装

无需安装，直接下载源码即可使用。

### 使用方法

在项目根目录下运行 `python todo.py` 加上相应的指令。

#### 1. 添加任务

```bash
python todo.py add "买牛奶"
python todo.py add "写代码"
```

#### 2. 查看任务列表

```bash
python todo.py list
```

输出示例：
```
📋 任务列表：
1. [❌] 买牛奶
2. [❌] 写代码
```

#### 3. 完成任务

支持标记单个或多个任务为已完成：

```bash
# 完成任务 1
python todo.py done 1

# 批量完成任务 1 和 2
python todo.py done 1 2
```

#### 4. 删除任务

```bash
# 删除编号为 1 的任务
python todo.py del 1
```

#### 5. 清理已完成任务

```bash
# 删除所有已标记为完成的任务
python todo.py clear
```

#### 6. 查看帮助

```bash
python todo.py --help
```

## 📂 数据存储

所有任务数据保存在当前目录下的 `todo.json` 文件中。
